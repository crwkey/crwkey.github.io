使用 Vercle + Github 搭建博客

Wed, Feb 2, 2022

前言

如何使你的 blog 秒开，今天我们使用Vercel来搭建咱们的博客，有几个步骤

  1. 注册 github, 地址： https://github.com/，这里不做过多赘述
  2. 注册 vercel

    1. 打开 https://vercel.com/signup 进行注册 点击 Continue with Github

    2. 弹出授权页面, 点击授权

    3. 选择 +86，然后输入你的手机号，点击 continue
